# AI Studio 앱 실행 및 배포

이 프로젝트는 앱을 로컬에서 실행하는 데 필요한 모든 요소를 포함하고 있습니다.

AI Studio에서 앱 보기: https://ai.studio/apps/049d680d-0b80-4f59-8933-195c52b4583b

## 로컬 실행 방법

**사전 요구 사항:** Node.js


1. 의존성 설치:
   `npm install`
2. [.env.local](.env.local) 파일의 `GEMINI_API_KEY`를 본인의 Gemini API 키로 설정합니다.
3. 앱 실행:
   `npm run dev`
